# Music Genre Classification using Machine Learning

## Project Overview
This project uses machine learning to classify songs into genres based on audio features like tempo, loudness, and energy.

## Algorithms Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Features Used
- Tempo
- Loudness
- Danceability
- Energy
- Acousticness
- Valence

## Results
The models were evaluated using accuracy, and SVM performed the best among all.

## How to Run
1. Install required libraries:
   pip install pandas numpy matplotlib scikit-learn

2. Run the code:
   python main.py
